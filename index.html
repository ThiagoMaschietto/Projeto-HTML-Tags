Primeiros passos com HTML

O que são as Tags

As tags são basicamente o meio com o qual temos como nos comunicar com a maquina, o inicio e o fim de um elemento que sera usado no HTML.

As tags definem o conteúdo e a formatação de um documento HTML.

Exemplos de tags

H1-H6 HR

As tags que vão do h1 até o h6, reprentam os titulos e os subtitulos de uma pagina podendo ir diminuindo de tamanho quanto maior o numero após o h menor será a fonte, e também tem o hr que serve para colocar uma linha na pagina.

P

A tag p reprenta um paragrafo de uma pagina.

Blockquote

A tag blockquote representa uma citação.

A

A tag a serve para definir um link para uma palavra.

Del

A tag del serve para deletar um texto.Exemplo

Tags de Personalização

Mark

A tag mark serve para marcar uma palavra.

Small

A tag small serve para diminuir uma palavra.

I

A tag i serve para deixar a palavra em italico.

U

A tag u serve para deixar a palavra sublinhada.

Font

A tag fonte era utilizada para definir a fonte, o tamanho e a cor de um texto, porém não é mais utilizada.

Abbr

A tag abbr serve para representar uma abreviação ou um acrônimo.EXP(Exemplo).

Tags de Posição
Sub

A tag sub serve para colocar a palvra abaixo das outras.

Sup

A tag sup serve para colocar a palvra acima das outras.

Tags de Listas

OL

A tag ol serve para montar uma lista ordenada.

UL

A tag ul serve para montar uma lista não ordenada.

LI

A tag li serve para listar os itens de uma lista seja ela ordenada ou não.

Exemplo:
  1. Tag ordenada 1
  2. Tag ordenada 2
  • Tag não ordenada 1
  • Tag não ordenada 2
